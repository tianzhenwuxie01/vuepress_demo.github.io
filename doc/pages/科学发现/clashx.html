    # ClashX

    Clash 是一款界面优雅的客户端，支持丰富的自定义规则：

    # 下载与安装

    guthub 地址：https://github.com/yichengchen/clashX/releases (opens new window)

    潮汐下载：https://www.lanzous.com/i9i1m1g (opens new window)

    # 安装

    双击下载好的clashx.dmg文件，在弹出的窗口中，将ClashX图标拖动到右侧的Application文件夹内即可完成安装。

    # 运行 ClashX

    打开启动台，找到 ClashX 的图标，单击即可打开。

    首次启动 clashx 会申请权限，点击安装并输入密码确定

    顶部菜单栏右上角便显示 ClashX 的程序图标了

    # 导入配置文件

    复制潮汐用户中内的 clash 订阅

    单机右上角 clashx 图标，依次选择配置-托管配置-管理

    将之前复制的订阅添加到 clashx 内

    点击 clashx 图标选择配置就能看见之前添加的订阅了

    # 开启系统代理

    点击 calshx 图标勾选设置为系统代理，顶部小猫图标会变亮

    出站模式选择规则

    点击Proxy可以选择你要连接的节点

    就可以愉快上网了
