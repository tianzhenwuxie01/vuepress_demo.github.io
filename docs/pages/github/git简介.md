## git：版本控制工具

## github：全球最大开源项目托管平台

##### 基本概念：

仓库（Repository）：用来存放开源项目代码的，有多少开源项目就有多少个仓库

收藏（star）：收藏项目，方便下次查看

复制/叉子（fork）：克隆别人的项目，再自己的账户里生成一个仓库

发起请求(Pull Request)：克隆别人仓库，自己修改了后强求原作者应用自己的修改

关注（watch）：了解别人动态

事务卡片(lssue）：发现别人问题或 bug，发起讨论
