#### shutdown 指令只能是超级管理员使用

shutdown -t 分钟数：多少分钟后关机（直接关机）
shutdown -t 分钟数：多少分钟后重启
shutdown -k 分钟数：多少分钟后关机（只发出关机警告，不关机）
shutdown -c 分钟数：取消关机。快捷键 ctrl+c
